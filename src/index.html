Обо мне

Далеко-далеко за словесными горами в стране гласных и согласных живут, рыбные тексты.

Меня зовут Александр. Живу в Могилеве (Беларусь). Занимаюсь HTML-версткой сайтов.

У меня кроссбраузерная (IE11, Chrome, Mozilla, Safari), адаптивная и валидная верстка.

Мои рабочие инструменты:

  • Gulp для автоматизации процесса верстки
  • Sass для ускоренной стилизации сайта и более гибкого кода
  • БЭМ naming
  • Bootstrap (в зависимости от проекта)
  • нативный JS и плагины Jquery
  • Wordpress в качестве движка
